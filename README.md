## HTML Basics Challenges

### 1. HTML Document Structure
```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <!-- Your content here -->
</body>
</html>
```
**Challenge**: Create a complete HTML5 document with a title "About Me" and add a heading that says "Welcome to My Page".

### 2. HTML Headings & Paragraphs
**Challenge**: Create a page with:
- A main heading (h1) "My Favorite Books"
- Three subheadings (h2) for different genres
- At least two paragraphs under each genre describing books you like

### 3. HTML Lists
**Challenge**: Create both an ordered and unordered list:
- Ordered list of your top 5 movies
- Unordered list of ingredients for your favorite recipe

### 4. HTML Links & Images
**Challenge**: Create a page with:
- A link to your favorite website that opens in a new tab
- An image from a local file
- An image from a web URL with proper alt text

### 5. HTML Tables
**Challenge**: Create a table representing a class schedule with:
- Columns for Day, Time, Subject, and Room
- At least 5 rows of data
- Appropriate table headers

## CSS Basics Challenges

### 1. CSS Colors & Backgrounds
```html
<div class="color-box"></div>
```
**Challenge**: Style the div to:
- Be 200px wide and tall
- Have a semi-transparent red background
- A 5px dashed blue border
- Rounded corners

### 2. CSS Text Styling
```html
<p class="fancy-text">This is some text to style</p>
```
**Challenge**: Make the text:
- Arial font, size 24px
- Italic and bold
- Letter spacing of 2px
- Shadow effect (2px horizontal, 2px vertical, 3px blur, grey color)

### 3. CSS Box Model
```html
<div class="box-model-demo">Content</div>
```
**Challenge**: Style the div to demonstrate the box model:
- 300px width with 20px padding
- 10px margin all around
- 5px solid border
- Center the box on the page

### 4. CSS Positioning
```html
<div class="parent">
    <div class="child1"></div>
    <div class="child2"></div>
</div>
```
**Challenge**: 
- Make parent 400px √ó 400px with relative position
- Position child1 absolutely in top-right corner
- Position child2 fixed to bottom of viewport

## Intermediate CSS Challenges

### 1. CSS Flexbox
```html
<div class="flex-container">
    <div>1</div>
    <div>2</div>
    <div>3</div>
</div>
```
**Challenge**: Style the container to:
- Display as flex row by default
- Center items vertically
- Space items evenly
- Wrap on smaller screens
- Make items grow/shrink appropriately

### 2. CSS Grid
**Challenge**: Create a 3√ó3 grid layout with:
- Header spanning all columns
- Sidebar in first column (rows 2-3)
- Main content in columns 2-3 (rows 2-3)
- Footer spanning all columns
- Make it responsive to collapse on mobile

### 3. CSS Transitions & Animations
```html
<button class="animated-button">Hover Me</button>
```
**Challenge**: Create a button that:
- Changes color smoothly on hover
- Grows slightly when clicked
- Has a pulsating animation when at rest

### 4. CSS Responsive Design
**Challenge**: Create a page that:
- Has a mobile-first design
- Uses media queries to adapt layout
- Adjusts font sizes based on viewport
- Shows/hides elements at different breakpoints
